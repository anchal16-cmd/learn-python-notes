## 🚀 How to Use

### Clone this repository

```bash
git clone https://github.com/anchal16-cmd/learn-python-notes.git
cd learn-python-notes
```

### Run any file

```bash
python filename.py
```

### Learn actively

* Read the comments carefully.
* Run the code yourself.
* Modify values and experiment.
* Break the code and understand why errors occur.

> 💡 The fastest way to learn programming is by writing, testing, and fixing your own code.
