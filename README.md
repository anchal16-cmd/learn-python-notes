## How to run any file
Terminal mein jaake type karo:
```
python filename.py
```
Example:
```
python 01_basics.py
```

## Tip
Har file ko run karke output dekho, fir comments padhke samjho ki output kyu aaya. Yeh sabse fast tareeka hai seekhne ka.
