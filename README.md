## 📌 About

This repository is a personal collection of Python notes, written while learning Python from scratch. Each file covers **one topic only**, so you can study, revise, or revisit any concept without digging through one giant file.

Every file is a runnable `.py` script — run it and see the output for yourself.

## 🚀 How to Use

1. **Clone this repository**
   ```bash
   git clone https://github.com/anchal16-cmd/learn-python-notes.git
   cd learn-python-notes
   ```

2. **Run any file**
   ```bash
   python 01_basics.py
   ```

3. **Read the comments** alongside the output — that's where the actual learning happens.

> 💡 Tip: Don't just read the code. Run it, change values, break it, and see what happens. That's the fastest way to actually learn.

## 🛣️ Suggested Learning Order

Basics → Operators → Strings → Conditionals → Loops → Lists/Tuples → Dictionaries/Sets → Functions → OOP → Exception Handling → File Handling → Modules

Following this order builds concepts on top of each other naturally.
